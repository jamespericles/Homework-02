Work

Below are some of the projects I'm most proud of. A link to each repo is provided.

Employee Management System GitHub

The Employee Management System is a CLI project that utilizes Node.js, Javascript, and MySQL to create a robust application to manage a population of employees. It offers several different functions, including the ability to not only create employees, roles, and departments, but also manipulate them after their creation.

I am very proud of this project, particularly the way I made it so that existing data can be retrieved and presented as choices where appropriate. This required me to nest several queries within one another. This project also taught me how to rebase a repository!

Random Password Generator GitHub

Choosing from four different groups of characters, users are able to create a password ranging from 8-128 characters in length. Besides displaying the generated password, the user may also copy them directly to their clipboard.

This project comes in two flavors! I was expected to build it using Javascript and basic HTML early on in my Bootcamp. Over a long weekend, I taught myself XCode and Swift by recreating this project over the course of three days. The repository for the Swift version of this project can be found here.

Read Me Generator GitHub

This CLI application utilizes Node.js and Inquirer to prompt the user with several questions related to their README file. Afterwards, the answers given are used to affect the document in several ways. Through the user of ternary operators, the user is able to delete unnecessary sections of the README. The document generates in a Github friendly way, including links to each section of the document.

I have ended up using this project for each assignment I've been assigned after its creation.
